Overview¶

This portal provides a detailed documentation of the MAMMOTH: Modular Adaptable Massively Multilingual Open Translation @ Helsinki.

Installation¶

pip install mammoth-nlp

Check out the installation guide to install in specific clusters.

Take a look at the quickstart to familiarize yourself with the main training workflow.

Citation¶

This project is based on OpenNMT V2. When using OpenNMT-py for research please cite the OpenNMT technical report

We published FoTraNMT the ancestor of MAMMOTH, when using MAMMOTH for research, please cite Boggia et al. (2023).
